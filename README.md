# *CloudStatus*
### 一个以实用为主的服务器玩家监视工具
没啥美术细胞

---
暂时不会为此软件提供任何许可证，软件代码仅供学习

---
### 我们提供
1. 可选日期可滚动的 ___玩家数量图表___
2. 可选日期的服务器玩家 ___在线时间分析___
3. 小巧的数据体积, 数据量: ___600KB/day___ (日均10玩家)
### 如何使用
1. 下载项目压缩包
2. 解压至某地并在终端中打开
3. `pip install -r requirements.txt` 安装对应运行库
4. `python main.py`运行项目
5. 项目运行后可在GUI界面配置服务器地址
### 运行环境
1. Python 3.10.9
